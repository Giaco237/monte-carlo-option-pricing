# Monte Carlo Option Pricing

## Objective
This project prices European options using Monte Carlo simulation and compares the results with the Black-Scholes model.

## Methodology
The project:
- simulates terminal stock prices under geometric Brownian motion,
- computes European call and put payoffs,
- discounts expected payoffs to obtain option prices,
- compares Monte Carlo prices with Black-Scholes prices,
- visualizes convergence and sensitivity to volatility.

## Models Used
- Monte Carlo Simulation
- Black-Scholes Model

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib
- SciPy

## Conclusion
This project demonstrates how Monte Carlo simulation can be applied to option pricing and validated against analytical Black-Scholes results.
